## 目标用户分群

### 小白用户（Home User / 入门级）
- 技术水平：无网络专业背景
- 典型场景：家庭网络搭建、小店铺 Wi-Fi 覆盖
- 核心诉求：开箱即用，清晰引导，出问题有明确提示
- 使用模式：Standalone 为主
- 痛点：术语困难、Controller 概念混淆

### Prosumer / 高级家庭用户
- 技术水平：有一定网络知识，愿意折腾但非专业
- 典型场景：多 AP 家庭覆盖、小型办公室、民宿
- 核心诉求：丰富功能但不需企业级复杂度，远程管理
- 使用模式：Controller 模式（云/软件 Controller）
- 痛点：功能入口不直观、VPN 配置复杂

### 专业工程师 / IT 管理员
- 技术水平：专业 IT 背景，管理多站点
- 典型场景：企业网络部署、连锁门店、MSP 服务商
- 核心诉求：批量操作效率、全面监控、App 与 Web 一致
- 使用模式：Controller 模式（硬件/云 Controller）
- 痛点：App 功能不如 Web 完整、缺少高级诊断

## 核心使用场景
1. 网络开通（首次部署）
2. 日常运维
3. 故障排查
4. 访客管理（Voucher / Portal）
5. 远程管理
