## Omada 产品生态定位
Omada 是 TP-Link 面向中小企业（SMB）及 Prosumer 市场的 **软件定义网络（SDN）管理平台**，提供集中化的网络设备管理能力，覆盖无线 AP、交换机、路由器、VPN 网关等全品类网络设备。
**Omada App** 是该平台的移动端入口，定位为：
- 网络开通与初始化的便捷工具
- 日常运维与监控的随身面板
- 故障排查与告警响应的即时通道

## 管理模式
### Controller 模式
- 通过 Omada SDN Controller（硬件/软件/云）集中管理所有设备
- 功能最全：站点管理、客户端管理、VPN、Portal、Voucher、高级网络配置等
- 适合多站点部署、专业 IT 管理场景

### Standalone 模式
- 直连单台设备进行管理，无需 Controller
- 功能受限于单设备能力：基础 Wi-Fi 设置、设备信息查看、固件升级等
- 适合家庭用户、极小型部署、临时调试

## 云服务层级
### Cloud Essentials（免费）
- 远程设备监控与管理、基础告警推送、固件远程升级

### Cloud Standard（付费订阅）
- 高级报告与分析、增强安全功能、AI 助手（规划中）

## 与 Omada Guard 的关系
- Omada App：网络设备管理为主
- Guard App：安防设备（摄像头、NVR）管理为主
- 长期方向：两者功能逐步融合（5.4+）
